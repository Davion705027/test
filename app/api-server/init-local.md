
## 技术栈
> server: egg + mongoose

> 16 <= node <= 18

## server

```bash
# 克隆项目(commit最近时间一般就是最新分支)
git clone http://git.sportxxxr1pub.com/front_web/client-api-doc-server

# 进入项目目录
cd client-api-doc-server

# 数据库初始化
cd ./database-mongodb/seeders

node init_admin

# 安装依赖
npm install

# 启动服务
npm run dev
```

本地服务地址：http://127.0.0.1:18101
